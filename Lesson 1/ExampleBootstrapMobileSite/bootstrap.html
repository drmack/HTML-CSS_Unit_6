What is Bootstrap?

In the CodeHS Web Design course, we are learning how to build websites using Bootstrap! Bootstrap is framework for developing responsive, mobile websites. It's like a toolbox for web developers.


Bootstrap

Bootstrap is one of the most popular frameworks for building professional, responsive, mobile websites. Millions of websites are built using Bootstrap, including CodeHS!

In the CodeHS Web Design course, students learn to use Bootstrap to develop their own beautiful websites that are viewable from any device.
